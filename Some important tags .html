Some important formulas of square

^ Formulas where square are involved are:

  • (a+b)2 = a2 + b2 + 2ab
  • (a-b)(a+b) = a2 -b 2
  • (a-b-c)2 = a2 + b2 + c2 - 2ab + 2bc - 2ca

^ Formulas where cube are involved are:

  • (a+b)3 = a3 + b3 + 3ab(a+b)
  • if (a+b+c) = 0,
    a3 + b3 + c3 = 3abc
  • a3 + b3 + c3 - 3abc = (a + b + c)(a2 + b2 + c2 - ab -bc - ac)

^ Power n Formula

  • an - bn = (a - b)(an-1 + an-2 b1 + an-3 b2 + ....+ a1 bn-2 + bn-1)
